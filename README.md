# üê¶ Flap and Fly

A fun, addictive, and simple "Flappy Bird" style game built entirely with vanilla web technologies. Challenge yourself to get the highest score!

![A screenshot of the Flap and Fly game in action, showing the bird, pipes, and score.](screenshot.png)

> **Note:** You'll need to add your own `screenshot.png` to the repository for the image to appear.

## ‚ú® Features

-   **Start & Game Over Screens:** A clean user interface to start the game and view your final score.
-   **High Score Tracking:** Your best score is saved locally in your browser, so you can always try to beat it!
-   **Smooth Animations:** The bird rotates realistically as it flaps and falls.
-   **Custom SVG Bird:** A simple but charming vector graphic bird.
-   **Fair Collision Detection:** The hitbox is slightly smaller than the bird itself for a more forgiving and enjoyable gameplay experience.
-   **Single File:** No complex setup! The entire game is contained in a single HTML file.
-   **Responsive Controls:** Use either a mouse click or the spacebar to play.

## üéÆ How to Play

1.  Click the **"Start Game"** button.
2.  **Click** anywhere on the game screen or press the **Spacebar** to make the bird flap its wings and fly up.
3.  Navigate through the gaps in the green pipes.
4.  The game ends if you hit a pipe or fly into the ground or ceiling.
5.  Try to beat your high score!

## üöÄ Getting Started

No installation or build process is required! Just follow these simple steps:

1.  Clone this repository to your local machine.
    ```bash
    git clone https://github.com/your-username/flap-and-fly.git
    ```
2.  Navigate into the project directory.
    ```bash
    cd flap-and-fly
    ```
3.  Open the `index.html` file in your favorite web browser (like Chrome, Firefox, or Edge).

That's it! The game will start, and you can begin playing.
